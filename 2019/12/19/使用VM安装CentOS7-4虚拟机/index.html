使用VM安装CentOS7.4虚拟机
发表于 2019-12-19| 更新于 2019-12-19|Linux基础
阅读量:

CentOS的官方站
http://www.centos.org
http://vault.centos.org
阿里云站点
http://mirrors.aliyun.com

安装

  1. 首先设置一下VM的网关, 子网ip及子网掩码

  1. 在VM中创建一个虚拟机

    注意

    • 在安装客户机操作系统时选择稍后安装操作系统
    • 版本选择选centos7 64位
    • 网络连接时选择使用网络地址转换(NET)

    其余部分选择默认即可

  2. 在设备的CD/DVD中为虚拟机选择镜像

  3. 开启虚拟机

  4. 选中第一个(选中的选项会变成白色)按tab键然后敲一个空格后输入指令net.ifnames=0, 敲回车

  5. 安装语言选择中文

  6. 时区选择上海

  7. 软件选择如图

  1. 点进网络与主机名, 打开, 再配置


  1. 开始安装
  2. 设置root密码, 等待安装完成
  3. 测试网络: ping baidu.com

配置(常用软件安装, 修改主机名, 网络, 关闭防火墙)

  1. 常用软件安装

    yum install -y bash-completion vim lrzsz wget expect net-tools nc nmap tree dos2unix htop iftop iotop unzip telnet sl psmisc nethogs glances bc

  2. 修改主机名

    hostnamectl set-hostname 新的主机名

  3. 网络

    echo "10.0.0.100 主机名">>/etc/hosts

  4. 关闭防火墙
